# RFO-macOS

A RFO installer for the macOS ecosystem.

JSON is provided by Kaiddd

# Installing

## First Install
1. Download rfo-install.sh
2. Test

## Renewing
1. Do this command:

```curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/54thTaco/RFO-macOS/main/rfo-install.sh | sh```
